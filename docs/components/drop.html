Basic

Drop component provide the same function as Bootstrap Dropdown and Popover. but it's more flexible, you can use is as dropdown menu, dropup form, dropup content and so on.

Position

The default position of Drop is drop bottomLeft. You can change the position by adding modifier classes below on .c-drop container element.

  • .c-drop-topLeft
  • .c-drop-top
  • .c-drop-topRight
  • .c-drop-rightTop
  • .c-drop-right
  • .c-drop-rightBottom
  • .c-drop-bottomLeft
  • .c-drop-bottom
